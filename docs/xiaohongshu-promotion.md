作为一个经常要连服务器的人，试过各种 SSH 工具，要么收费、要么广告满天飞、要么功能臃肿到用不上。

花了点时间自己撸了一个 SSH 桌面工具 **Easy Shell**，用下来感觉就是：**该有的全有，没有的全不需要。**

---

## ✨ 为什么推荐它？

**🖥️ 真终端体验**

不是那种假终端，是真的 SSH 连接 + 终端模拟器。多标签页同时开好几个会话，窗口大小变了终端自动跟着调，跟自带终端一样丝滑。

**📂 文件管理太方便了**

连上服务器后，右侧直接弹出文件面板：
- 看文件列表、上传下载、新建文件夹
- 支持拖拽上传（直接把文件拖进去就行）
- 内置快捷目录跳转：一键切到 `/opt`、`/var/log`、`/etc` 这些常用路径
- 还能输任意路径直接跳转

**📊 服务器信息一目了然**

连上之后自动采集系统信息，直接在列表里就能看到：
- 什么系统（Ubuntu / CentOS / Debian…）
- 几核 CPU
- 内存多大
- 硬盘多大

不用登进去再手动 `df -h` 了。

**🎨 5 套主题随便切**

暗夜、纯黑、深蓝、明亮、纸白，5 套配色方案。终端配色跟着主题走，深色浅色都有，眼睛舒服了。

**🔒 数据全在本地**

连接信息只存在你自己电脑上，不上传任何数据到云端。对安全有要求的同学可以放心用。

## 📥 怎么下载？

GitHub 开源项目，完全免费：

👉 **下载地址：https://github.com/EasyRoc/easy-shell/releases**

支持的系统：
- **Mac（M芯片）**：选 `arm64.dmg`
- **Mac（Intel芯片）**：选 `x64.dmg`
- **Windows 64 位**：选 `win-x64.zip`
- **Windows 32 位**：选 `win-ia32.zip`

> 💡 不知道自己什么芯片？Mac 点左上角苹果 → "关于本机"，看处理器是 "Apple" 还是 "Intel" 就行。
